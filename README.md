  ## À Propos

### Contexte du Projet

Cette application Power BI a été développée dans le cadre d’un **projet d’analyse des DPE** pour le département du **Rhône**, en partenariat entre l’**IUT** et **Enedis**.

Elle permet de visualiser finement la performance énergétique des logements, de repérer les **passoires énergétiques** (étiquettes F et G)
