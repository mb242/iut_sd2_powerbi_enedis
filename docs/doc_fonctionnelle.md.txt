# Documentation Fonctionnelle  
## Observatoire DPE du Rhône

---

## 1. Objectifs de l’application

Cette application Power BI permet aux acteurs publics et décideurs locaux de :

- Comprendre la **performance énergétique des logements**
- Identifier les **zones à forte concentration de passoires thermiques**
- Comparer les consommations, émissions et coûts
- Orienter les **politiques de rénovation énergétique**

---

## 2. Public cible

- Collectivités territoriales
- Services urbanisme & énergie
- Préfecture
- ENEDIS
- Étudiants & analystes

---

## 3. Navigation dans l’application

### Pages disponibles

| Page | Description |
|----|------------|
| Vue d’ensemble | Indicateurs clés globaux |
| Performance DPE | Répartition DPE & GES |
| Analyse | Coûts, surfaces, consommation |
| Carte | Analyse géographique |

Navigation via menu latéral.

---

## 4. Filtres interactifs

Les segments permettent un filtrage dynamique :

- Commune
- Type de logement
- Type d’énergie
- Étiquette DPE
- Période de construction

Les filtres impactent **l’ensemble des visuels**.

---

## 5. Indicateurs clés (KPI)

- Nombre de logements
- Nombre de communes
- % logements passoires (F-G)
- % logements performants (A-C)
- Émissions moyennes de CO₂
- Coût moyen du chauffage
- Surface habitable moyenne

---

## 6. Visualisations principales

### Histogrammes
- Répartition DPE (A → G)
- Répartition GES

### Donuts
- Logements par type de bâtiment
- Logements par type d’énergie

### Nuage de points
- Surface habitable vs coût de chauffage
- Consommation vs émissions

---

## 7. Cas d’usage

- Identifier les communes prioritaires
- Comparer logements anciens / récents
- Étudier l’impact du chauffage électrique
- Analyser les coûts énergétiques

---

## 8. Sécurité et accès

Selon le rôle utilisateur :
- Vue départementale complète
- Vue restreinte par département (RLS)

---

## 9. Bénéfices

- Lecture rapide de l’état énergétique
- Outil d’aide à la décision
- Vision synthétique et détaillée
- Données fiables et interactives
